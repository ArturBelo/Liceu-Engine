## Visão

O **Liceu Engine** é um sistema de engenharia de conhecimento projetado para construir, organizar e evoluir um ecossistema pessoal de aprendizagem.

Seu objetivo é acompanhar o usuário durante toda a vida acadêmica e profissional, mantendo o conhecimento estruturado, pesquisável e independente da plataforma utilizada.

---

## Missão

Construir um software capaz de organizar conhecimento de forma permanente.

---

## Princípios

- Organização acima da complexidade.
- Independência de plataforma.
- Automação sempre que possível.
- Código limpo e documentado.
- Evolução contínua.

---

## Arquitetura

O núcleo da Engine nunca dependerá de uma plataforma específica.

As integrações serão implementadas como módulos independentes.

Plataformas planejadas:

- Notion
- Obsidian
- Markdown
- Futuras integrações
